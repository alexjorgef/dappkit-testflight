@taiki/dappkit testflight Test application for dappkit proxies

Deploy and Interact with web3 smart contracts

@taikai/dappkit-testflight is a dapp used to test and deploy smart contracts via a web browser.

It achieves this by using @taikai/dappkit as a base for smart contract interaction

How to

  • Open sidebar
  • Search for and connect to a EVM
  • Select (or load a custom ABI) and interact with a Contract
